# ET 핵심 기능 최적화 결과 분석

## 최종 판정

신뢰도와 추천 모두 1차 최적화 변경을 채택할 근거가 확보됐다.

- 신뢰도: W2 가중치 + evidence 4점 미만 69점 상한, 불변쌍 9/9 통과.
- 추천: 임베딩 후보에 category/trust/freshness를 결합한 hybrid 재정렬 채택.
- 회귀 테스트: 관련 테스트 및 smoke test 36개 통과.

내부 신호만 사용한 서울대 팩트체크 Spearman ρ=0.391은 목표 0.5에 미달한다. 외부 검색 점수를 20% 혼합한 전체 30건 진단은 ρ=0.552였지만, train fold에서만 혼합 비율을 선택한 5-fold out-of-fold 값은 0.462였다. 따라서 현재 파이프라인을 “사실 확인 완료 모델”로 표현하면 안 되며, 기본 기능은 “기사 내부 신뢰 신호를 이용한 위험도 추정”, 외부 검색은 “실험적 주장 교차검증”으로 설명하는 것이 정확하다.

## 신뢰도 최적화 결과

### 채택 변경

```text
source 0.20
evidence 0.20
style 0.25
logic 0.15
clickbait -0.20
```

- evidence predicate에 독립 출처 수와 추적 가능한 1차 자료·연구명을 추가.
- logical predicate에 상반된 인용과 기사 서술자의 자기모순 구분을 추가.
- evidence < 4이면 최종 점수를 최대 69점으로 제한.
- 인간 관점에서 모순이 아니었던 P5 fixture를 직접 자기모순 문장으로 수정.

### 성능

| 지표 | 기존 W3 | 채택 W2 | 변화 |
|---|---:|---:|---:|
| 불변쌍 통과율 | 8/9 | 9/9 | +1건 |
| SNU Spearman ρ | 0.308 | 0.391 | +26.9% |
| 극단 레이블 AUC | 0.710 | 0.775 | +9.2% |
| 고·저신뢰 평균 격차 | 10.7점 | 15.7점 | +46.7% |

`trust_eval_samples.jsonl`은 낚시성 분류 데이터라 사실성 평가에서 제외했다. 이 세트의 역전 결과는 별도 태스크 불일치 증거로만 보존한다.

### 외부 검색 교차검증 후보

| 지표 | W2 내부 신호 | 5-fold OOF 혼합 | 전체 30건 alpha=0.2 진단 |
|---|---:|---:|---:|
| Spearman ρ | 0.391 | 0.462 | 0.552 |
| 극단 레이블 AUC | 0.775 | 0.800 | 0.860 |

전체 데이터 진단값은 가중치 선택과 평가에 같은 표본을 사용하므로 최종 성능으로 인용하지 않는다. 운영 기본값은 여전히 W2이며, grounding은 명시적 `--live`에서만 실행되는 오프라인 후보로 유지한다.

## 추천 최적화 결과

### 평가 신뢰성 수정

- 클릭 이후 impression이 negative에 포함되던 미래 정보 누수 제거.
- 30초 내 click/view 중복 positive 967건 제거.
- 현재 임베딩이 없는 삭제 기사 후보 제거.
- 점수 동률일 때 positive-first 입력 순서가 유지되던 편향 제거.
- random 기준선을 SHA-256 seed로 재현 가능하게 변경.

### 채택 변경

임베딩 검색 후보를 다음 점수로 재정렬한다.

```text
semantic 0.10 + category 0.60 + trust 0.10 + freshness 0.20
```

임베딩은 후보 검색 단계에서 계속 핵심 역할을 한다. 재정렬은 Render에서도 torch 없이 수행 가능한 경량 연산이다.

프로필 경로에서는 learned/raw 공간 혼합을 제거했다. 동일 기사 21건의 learned/raw 코사인 평균은 0.173이었고, learned 벡터로 raw chunk의 자기 기사를 찾는 Top-1은 23.8%뿐이었다. raw-only 프로필은 기존 mixed 프로필 대비 실제 로그 MRR을 0.5530에서 0.6942로, NDCG@3를 0.5957에서 0.7160으로 회복했다.

### 실제 시간 순서 로그 118건

| 지표 | profile | hybrid | 변화 |
|---|---:|---:|---:|
| HitRate@3 | 0.8729 | 0.8814 | +0.85%p |
| MRR | 0.6942 | 0.7458 | +7.4% |
| NDCG@3 | 0.7160 | 0.7584 | +5.9% |

### 인간 직관 페르소나 5-seed 검증

총 validation 2,640건에서:

| 지표 | semantic-only | hybrid | 변화 |
|---|---:|---:|---:|
| NDCG@3 | 0.5231 | 0.6310 | +20.6% |
| MRR | 0.5420 | 0.6193 | +14.3% |
| 저신뢰 Top-3 비율 | 1.439% | 0.922% | -36.0% |

페르소나 정의와 행동 가정은 `Persona_Benchmark_Spec.md`에 명시했다.

## 보고서에 사용할 수 있는 문장

> 평가 데이터의 시간 누수와 중복 행동을 제거한 뒤, 실제 로그 118건에서 임베딩 단독 추천 대비 인간 직관 기반 hybrid 재정렬의 MRR이 7.4%, NDCG@3가 5.9% 향상됐다. 8개 페르소나를 적용한 5개 seed 합성 검증에서도 NDCG@3가 평균 20.6% 향상됐으며 저신뢰 기사 Top-3 노출률은 36.0% 감소했다. 신뢰도 산출은 근거 세분화와 W2 가중치, evidence 상한을 적용해 불변 방향성 9/9를 만족했고, 서울대 팩트체크 30건에서 기존 대비 Spearman 상관이 0.308에서 0.391로 개선됐다. 외부 검색 교차검증 혼합은 5-fold out-of-fold에서 0.462까지 개선됐지만 목표 0.5를 재현하지 못했으므로 오프라인 후보로 유지한다.

## 남은 핵심 과제

1. 외부 검색·RAG 근거 대조를 100건 이상 시간 분할 holdout으로 확대해 사실성 상관 ρ≥0.5를 재현.
2. 임베딩 공간 정합성 검증 및 분리 완료: stage-2 프로필은 raw `embed_summary`, Attention은 learned 공간만 사용한다.
3. Render에서 hybrid p95 latency와 `rec_source`별 CTR 온라인 검증.
4. 실제 temporal 리플레이 300건, 팩트체크 100건 이상으로 신뢰구간 산출.

세부 가설·실험·채택/기각 이력은 `Optimization_Journal.md`를 기준으로 한다.
